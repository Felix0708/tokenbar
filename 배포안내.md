# TokenBar 배포 가이드 (무료 배포)

## 1. 패키지 만들기

```bash
cd ~/Claude/토큰조사/TokenBar
./package.sh
```

→ `dist/TokenBar-mac.zip` 생성 (앱 + 받는 사람용 설치방법.txt 포함)

## 2. 공유하기

**간단하게**: zip을 카톡, 메일, 구글 드라이브 등으로 그대로 전달.

**GitHub Releases (추천 — 링크 하나로 배포)**:

1. github.com 에서 새 저장소 생성 (예: `tokenbar`, Public)
2. 저장소 페이지 → 오른쪽 **Releases** → **Create a new release**
3. Tag: `v1.0.0` 입력 → **Publish** 전에 zip을 드래그해서 첨부
4. **Publish release** → 생성된 다운로드 링크를 공유

소스 코드도 같이 올리고 싶으면:

```bash
cd ~/Claude/토큰조사/TokenBar
git init && git add -A ':!build' ':!dist' && git commit -m "TokenBar v1.0"
# GitHub에서 만든 저장소 주소로:
git remote add origin https://github.com/내아이디/tokenbar.git
git push -u origin main
```

## 3. 받는 사람이 알아야 할 것

- Apple 공증이 없어서 첫 실행 시 "확인되지 않은 개발자" 경고가 뜸
  → 시스템 설정 → 개인정보 보호 및 보안 → **그래도 열기**
  (이 안내는 zip 안 `설치방법.txt`에 들어 있음)
- Claude Code / Codex CLI를 쓰는 Mac이어야 데이터가 표시됨
- macOS 14 (Sonoma) 이상, Apple Silicon/Intel 둘 다 지원

## 나중에 제대로 배포하려면

사용자가 늘면 Apple Developer Program($99/년) 가입 → Developer ID 서명 + 공증(notarization) 추가.
경고 없이 설치되고 위젯 등록도 안정적입니다. 필요할 때 공증 자동화 스크립트를 만들어 달라고 하면 됩니다.
